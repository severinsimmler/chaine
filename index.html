A Lightweight Conditional Random Field

This is a modern and fast Python library without any third-party dependencies implementing a linear-chain conditional random field for natural language processing tasks like named entity recognition or part-of-speech tagging.

You can install the latest stable version from PyPI: 

$ pip install chaine

In case you are interested in the theoretical concepts behind conditional random fields or the optimization algorithms used in this library, I can recommend the following papers:

  • Conditional Random Fields: Probabilistic Modelsfor Segmenting and Labeling Sequence Data
  • Scalable Training of L1-Regularized Log-Linear Models
  • Adaptive Regularization of Weight Vectors
